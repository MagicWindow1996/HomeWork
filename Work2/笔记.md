3. 数组
- map 
```
映射 [a,b,c,d] => [A,B,C,D]
从一个数据源 映射到另一个数据源 数据长度不变

arr.map(item=>item*2);
```
- reduce
```
汇总 [...arg] => x;
将数据源汇总为一个数
其两个参数分别为 上一个reduce的结果， 和当前数值.  其总是从下标为1 开始执行,所以其第一步时， temp = 12
    let arr= [12,5,8,99,27];

              let result = arr.reduce((temp,curr)=>{
              alert(temp);
                return temp + curr;
               });

```
- filter
```
过滤  进入一个数据源 通过条件过滤出另一个数据源

            <script>
                let arr= [12,5,8,99,27];

              alert( arr.filter(item=>item%2));
            </script>
```
- forEach
```
遍历 将数据源中所有数据遍历一遍
 let arr= [12,5,8,99,27];

arr.forEach((item,index)=>{alert(`第${index}个是${item}`)})

```

6.promise
```
专门用来解决异步操作的
操作步骤
1.创建Promise对象
  let p = new Promise(function (resolve, reject) {
            //resolve  成功
            //reject  失败
            $.ajax({
                url: "../data/1.txt",
                dataType: 'JSON',
                success(arr) {
                    resolve(arr);
                },
                error(arr) {
                    reject(arr);
                }
            })
        });

2.调用Promise对象内部的方法
  p.then(arr => alert("获取成功" + arr), arr => alert("失败了"));

或者拥有多个Promise对象 
  let p2 = new Promise(function (resolve, reject){...}
    let p3 = new Promise(function (resolve, reject){...}

将这些异步方法进行同步调用：

   Promise.all([p,p2,p3]).then(arr=>{
       // arr 保存的是3个Promise对象返回值的数组
                console.log(arr);
            },arr=>{})
```
